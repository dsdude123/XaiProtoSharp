# Security Policy

## Disclaimer

XaiProtoSharp is an auto-generated library with **no guarantee of functionality**. All library code is generated from upstream protobuf definitions in [xai-org/xai-proto](https://github.com/xai-org/xai-proto).

## Supported Versions

Only the latest published version of the package receives updates. Older versions are not patched.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public issue.
2. Email the maintainer or use [GitHub's private vulnerability reporting](https://github.com/dsdude123/XaiProtoSharp/security/advisories/new).
3. Include a description of the vulnerability and steps to reproduce it.

For security issues in the upstream protobuf definitions or the xAI API itself, please report them to [xai-org/xai-proto](https://github.com/xai-org/xai-proto) or xAI directly.
